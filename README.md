# Java RPG

## Short description

A simple, object-oriented RPG game created as a university project.  
The game demonstrates the basics of Java OOP, interaction between classes, combat mechanics, and game logic.

## Features

- Player creation
- Enemies with different stats
- Combat mechanics (attack, health, damage)
- Random encounters
- Input-based gameplay

## Technologies

- Java
- OOP principles
- Randomization
- Console I/O

## What I learned

- Building game logic step-by-step
- Designing cooperating classes
- Managing state and interactions
- Basic architecture planning

## How to run

1. Import the project into **IntelliJ**, **VS Code**, or **Eclipse**
2. Run `Main.java`
